## Dependencias
```
python 3.9
pipenv
```
## Instalação
Se você tiver o pip instalado executar o seguinte comando
```
pip install pipenv
```

## Instalando packages do projeto
_Executar comandos no shell dentro da pasta do projeto_
```
pipenv install
```


## Executando o projeto
_Executar comandos no shell dentro da pasta do projeto_

```
pipenv shell
python main.py
```
